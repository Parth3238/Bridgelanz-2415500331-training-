# Lambda Expressions & Method References - Solutions

This repository contains Java example code demonstrating:
- Lambda expressions for smart home lighting patterns, custom sorting, and notification filtering.
- Method references for printing patient IDs, uppercasing names, and constructor references for invoices.

Project structure:
- src/com/example/lambdas/*.java  -- Java source files
- .gitignore
- LICENSE (MIT)

How to compile & run:
1. Ensure Java JDK (11+) is installed.
2. From the project root run:
   ```
   javac -d out src/com/example/lambdas/*.java
   java -cp out com.example.lambdas.Main
   ```
The `Main` class runs all demonstrations and prints results to console.
